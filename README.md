# ✦ AcreFi — Where Money Grows

AcreFi is a modern, responsive landing page ecosystem designed for a programmable, utility-driven stablecoin. This repository contains the front-end architecture, a deep-dive protocol exploration layout, and an onboarding interface for the protocol's interactive beta platform.

## 🚀 Live Demo Pages
* **Landing Page (`index.html`)**: The primary gateway showcasing USD Bloom's features, strategic allocation layers, metrics, and core use cases.
* **Protocol Deep Dive (`explore.html`)**: An informative layout explaining mechanisms like Auto-Compounding and the Peg Stability Module.
* **Launch Beta (`beta.html`)**: A sleek, dark-themed Web3 wallet connection interface built with glassmorphic UI panels.

---

## ✨ Features

- **Cohesive Design System**: Styled seamlessly across all sub-pages using a unified typography framework (`Sora` for headers, `Inter` for prose).
- **Responsive Layouts**: Fully fluid grids and navigation breakpoints optimized across mobile, tablet, and desktop viewports.
- **Modern UI Components**: Features glassmorphism panels, interactive status chips, gradient glow backgrounds, and mobile-friendly hamburger menu toggles.
- **DeFi-Centric UX**: Engineered with a premium dark-aesthetic mode tailored for high-fidelity fintech and crypto applications.

---

## 🛠️ Tech Stack

* **Structure:** HTML5 (Semantic architecture)
* **Styling:** CSS3 (Flexbox, CSS Grid, Custom Properties, Backdrop Blurs)
* **Interactions:** JavaScript (ES6+ for state management and layout toggles)
* **Fonts:** Google Fonts (Sora & Inter)
